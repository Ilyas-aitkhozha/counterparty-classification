# Классификация и граф контрагентов

**Часть 2: Ветка A — Data / ML**

## Входные файлы

Перед запуском положите выданные файлы в папку `data/`:

```
data/transactions.csv
data/categories.json
data/gold_set.csv
```

## Запуск

```bash
pip install -r requirements.txt
jupyter notebook
```

Затем откройте ноутбуки в папке `notebooks/` и выполните `Restart & Run All`:

- `notebooks/eda.ipynb` — EDA и очистка (Часть 1.1–1.2)
- `notebooks/sql.ipynb` — SQL-запросы (Часть 1.3)
- `notebooks/classification.ipynb` — классификация (Часть 2A)
